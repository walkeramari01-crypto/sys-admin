## Tasks in this repo

- trash-recovery: macOS deleted-file recovery practice + notes/scripts
- sudoers-admin-group: macOS users/groups + sudo access practice (dseditgroup, visudo verification)
- bulk-user-creation: This task simulates a real IT/admin workflow: creating many employee accounts automatically instead of manually clicking "New User" in Active Directory. 
- least-privilege: This exercise simulates a security-first approach to network management.
